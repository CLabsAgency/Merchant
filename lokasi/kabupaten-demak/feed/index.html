    • ]]>         Microsoft Office Home & Student 2013 https://mp.velocitydeveloper.com/iklan/microsoft-office-home-student-2013/ Tue, 11 Aug 2015 06:46:53 +0000 https://mp.velocitydeveloper.com/iklan/microsoft-office-home-student-2013 Microsoft Office Home & Student 2013 merupakan paket software produktivitas yang ditujukan untuk pengguna rumahan atau pelajar. Terdiri dari Microsoft Word, Microsoft Excel, Microsoft PowerPoint dan Microsoft OneNote untuk 1 PC. Diciptakan untuk meningkatkan efektivitas dengan fitur yang kaya namun dengan tampilan modern yang mudah. Selain itu, paket software ini memuat fitur cloud computing yang dapat Anda akses dimana saja.

    Paket Aplikasi Office
    Microsoft Office Home & Student 2013 memuat paket Word 2013, Excel 2013, PowerPoint 2013 dan OneNote 2013 yang memiliki berbagai fitur baru yang mempermudah setiap pekerjaan.

    Microsoft Word 2013
    Pada aplikasi Word, Anda dapat membuka dan mengedit file PDF, membuka dan menonton video online tanpa meninggalkan aplikasi serta fitur Read Mode untuk tampilan yang kompatibel dengan berbagai layar.

    Microsoft Excel 2013
    Nikmati cara baru mengeksplorasi data secara intuitif. Dengan desain template yang baru. Fitur Flash Fill mampu mempelajari dan mengenali pola pengisian data Anda tanpa memerlukan formula atau macro.

    Microsoft PowerPoint 2013
    Ciptakan presentasi yang menakjubkan dengan alignment baru, color-matching dan fitur lainnya serta bagikan semuanya dengan cloud computing.

    Microsoft OneNote 2013
    Gambar, tulis tangan, ketik, klik dan geser notes pada Microsoft OneNote 2013. Notes multi-media dapat Anda simpan, cari dan sinkronisasi pada aplikasi OneNote di perangkat lainnya.

    Penyimpanan Cloud SkyDrive 7 GB
    Setiap dokumen, foto, presentasi dan note dapat disimpan secara otomatis melalui penyimpanan cloud SkyDrive. Anda dapat mengakses, mengedit dan menyimpannya dari perangkat internet dengan browser yang kompatibel.
